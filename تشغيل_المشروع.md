اولا انتقل الى مسار المشروع مثل:
```bash

cd /home/ramy/Desktop/Desktop/Projects/FlaskProjects/My_public_project

```
ثم فعل البيئه المعزوله ولو مش موجوده فقوم انشئ واحده جديده وثبت فيها فلاسك:
```bash

. .venv/bin/activate

```
## الجزئيه اللي تحت حق التشغيل في وضع خادم التطوير 
flask --app Ramy_Store run --debug --host=0.0.0.0
